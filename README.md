# IMP - Project

> Simple imperative language

Implementation of simple compiler with the following functionalities:

1. Type checker
2. Parser
3. Interpreter

- Based on: https://sulzmann.github.io/ModelBasedSW/imp.html
- See lecture notes

## Usage

1. Run project: `go run main.go`
2. Run tests: `go test -v`
